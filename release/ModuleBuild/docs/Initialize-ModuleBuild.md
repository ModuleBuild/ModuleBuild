# Initialize-ModuleBuild

## SYNOPSIS
Set up the framework for a ModuleBuild project.

## SYNTAX

```
Initialize-ModuleBuild [[-Path] <String>]
```

## DESCRIPTION
Set up the framework for a ModuleBuild project.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Initialize-ModuleBuild -Path C:\Work\NewModule
```

## PARAMETERS

### -Path
Specifies the path to create the project.
