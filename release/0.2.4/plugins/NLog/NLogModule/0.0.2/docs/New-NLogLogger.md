# New-NLogLogger

## SYNOPSIS
Creates a new LogManager instance

## SYNTAX

```
New-NLogLogger [-LoggerName] <String>
```

## DESCRIPTION
Important to log messages to file, mail, console etc.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
$myLogger = New-NLogLogger
```

## PARAMETERS

### -LoggerName
Name of the logger to get
