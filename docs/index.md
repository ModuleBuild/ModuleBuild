## Installation
ModuleBuild is available on the [PowerShell Gallery](https://www.powershellgallery.com/packages/ModuleBuild/).

To Inspect:
```powershell
Save-Module -Name ModuleBuild -Path <path>
```
To install:
```powershell
Install-Module -Name ModuleBuild -Scope CurrentUser
```
